🔧 사용 방법

1. WooCommerce 상품 생성

  1. WooCommerce → 상품 → 새로 추가
  2. 상품 데이터 섹션에서 "연결된 강의들" 선택 (Ctrl+클릭으로 다중 선택)
  3. 패키지 유형 선택 (번들/구독형/평생소장)
  4. 수강 기간 설정
  5. 자동 수강 등록 체크

2. 패키지 상품 표시

아래 Shortcode를 페이지나 포스트에 추가하세요:

[lectus_packages limit="12" columns="3" type="bundle"]

파라미터:

  • limit: 표시할 상품 수 (기본: 12)
  • columns: 열 개수 (기본: 3)
  • type: 패키지 유형 (bundle/subscription/lifetime)
  • category: 상품 카테고리 slug
  • orderby: 정렬 기준 (date/title/price)
  • order: 정렬 순서 (ASC/DESC)

🎯 주요 기능

패키지 상품 카드에 표시되는 정보

  • 패키지 타입 배지 (번들/구독형/평생소장)
  • 포함된 강의 목록
  • 총 강의 수 및 레슨 수
  • 예상 수강 시간
  • 수강 기간
  • 가격 정보 (할인가 포함)
  • 수강 상태 표시 (수강중/미수강)

자동화된 기능

  • 결제 완료 시 선택된 모든 강의에 자동 등록
  • 이미 수강중인 강의 체크 및 알림
  • 환불 시 자동 수강 취소
  • 주문 노트에 등록 내역 자동 기록

🧪 테스트 시나리오

다음 시나리오를 테스트하려면 WordPress 관리자 페이지에 로그인하세요:

관리자 페이지 열기 새 상품 만들기
  1. 패키지 상품 생성
    • WooCommerce 상품 생성
    • 여러 강의 선택
    • 패키지 타입 설정
  2. 상품 페이지 확인
    • 강의 정보 탭 표시
    • 포함된 강의 목록 확인
    • 패키지 요약 정보 확인
  3. 구매 프로세스
    • 장바구니 추가
    • 이미 수강중인 강의 체크
    • 결제 진행
  4. 자동 등록 확인
    • 주문 완료 후 강의 접근 확인
    • 내 강의 페이지에서 확인
    • 진도 추적 가능 여부
